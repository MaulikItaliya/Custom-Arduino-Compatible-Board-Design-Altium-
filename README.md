# Arduino Compatible 28-Pin Development Board

## Overview
Custom Arduino-compatible PCB designed in Altium Designer.

Features:
- ATmega328P-based design
- USB/UART programming
- 5V and 3.3V variants
- Through-hole + SMD support
- 2-layer PCB

## Tools Used
- Altium Designer
- Gerber Generation
- Assembly Drawings
- BOM Export

## Repository Structure
/docs
/hardware

/images

## Variants
### Variant 1
Fixed 5V version

### Variant 2
5V + 3.3V selectable version

Gerber, drill, BOM, pick-and-place, and assembly outputs included.
